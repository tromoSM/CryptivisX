# CryptivisX

CryptivisX is a Windows desktop application for encrypting and decrypting files locally using strong symmetric encryption.  
It uses a modern HTML, CSS, and JavaScript interface powered by Python and pywebview.

No cloud, no uploads, everything stays on your machine.

## Features

- File encryption and decryption
- Secure one time key system
- Proprietary encryption format
- Files encrypted by CryptivisX can **only** be decrypted by CryptivisX
- Other apps cannot decrypt the files even if the correct key is provided
- Image and video support
- QR code key export
- Modern desktop UI
- Runs fully offline
- Windows native app
